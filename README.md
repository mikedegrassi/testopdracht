# Testopdracht Invitado

## Opdracht
Maak een webformulier met minimaal de volgende velden/inputs:
- Aanhef (heer/mevrouw)
- Voornaam
- Tussenvoegsel
- Achternaam
- E-mailadres
- Landenkiezer

De ingestuurde resultaten van het formulier moeten:
- worden opgeslagen in een MySQL database
- beschikbaar worden gesteld via een JSON endpoint

## Randvoorwaarden
- Geen frameworks
- PHP als backend
- Code beschikbaar in een Git repository

## Beoordeling
De code wordt beoordeeld op:
- Compleetheid van de opdracht
- Structuur en leesbaarheid van de code  
- HTML/CSS-opmaak is van ondergeschikt belang
